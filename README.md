# Resume Intake Automation

A lightweight automation workflow for **resume screening and intake** using Zapier (or similar tools like n8n/Make).  
This project helps recruiters and HR teams streamline candidate intake by automatically collecting, parsing, and organizing resumes.

---

## 🚀 Features
- **Resume Collection**: Capture resumes via email, forms, or uploads.
- **Parsing & Data Extraction**: Automatically extract candidate details (name, email, skills, education).
- **ATS/Database Integration**: Push structured candidate data into Airtable, Google Sheets, or an ATS.
- **Notifications**: Send Slack/Email alerts when a new candidate is added.
- **Follow-Up Tasks**: Create recruiter reminders for next steps.

---

## ⚙️ Workflow Overview
1. **Trigger**: Candidate submits resume (via Gmail, Google Form, or Typeform).  
2. **Action**: Resume is parsed using a Resume Parser API or Zapier Formatter.  
3. **Action**: Candidate details are stored in Airtable/Google Sheets/ATS.  
4. **Action**: Recruiter notified via Slack/Email.  

---

## 🛠️ Tools & Integrations
- **Zapier / n8n / Make** – workflow automation  
- **Google Forms / Gmail** – resume intake  
- **Airtable / Google Sheets / ATS** – candidate database  
- **Slack / Email** – recruiter notifications  

---

## 📌 Getting Started
1. Clone this repository.  
2. Set up your Zapier (or n8n/Make) account.  
3. Configure triggers (resume intake) and actions (parsing, storage, notifications).  
4. Test with sample resumes.
