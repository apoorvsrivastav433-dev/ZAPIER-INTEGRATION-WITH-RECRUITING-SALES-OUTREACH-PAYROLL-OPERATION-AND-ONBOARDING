🔌 Complete API Reference

1. Google Forms Webhook - Form Submission

POST /hooks/zapier/catch/YOUR_WEBHOOK_ID
curl -X POST https://hooks.zapier.com/hooks/catch/WEBHOOK_ID \ -H "Content-Type: application/json" \ -d '{ "candidate_name": "Alice Johnson", "email": "alice@example.com", "phone": "+1-555-0123", "resume_url": "https://example.com/alice-resume.pdf", "position": "Senior Software Engineer", "submission_time": "2024-01-15T14:30:00Z", "source": "careers_form" }'

Response: Zapier receives and forwards to CandidateZip Parser

2. CandidateZip Resume Parser API

POST https://api.candidatezip.com/v1/parseResume
curl -X POST https://api.candidatezip.com/v1/parseResume \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "candidate_name": "Alice Johnson", "document_url": "https://example.com/alice-resume.pdf", "document_type": "resume", "return_fields": [ "name", "email", "phone", "location", "experience", "education", "skills", "certifications", "salary_expectation" ] }'

Returns: Parsed candidate data with confidence score

3. Screening Logic - Pass/Fail Decision

POST /hooks/zapier/catch/SCREENING_WEBHOOK
curl -X POST https://hooks.zapier.com/hooks/catch/SCREENING_ID \ -H "Content-Type: application/json" \ -d '{ "submission_id": "sub_12345", "candidate": { "name": "Alice Johnson", "email": "alice@example.com", "experience_years": 5, "skills": ["Python", "JavaScript", "React", "AWS"], "education": "Master in Computer Science", "location": "San Francisco, CA", "salary_min": 150000, "salary_max": 200000 }, "screening": { "score": 92, "recommendation": "STRONG_PASS", "passed": ["experience", "skills", "education", "location"], "failed": [] } }'

Triggers: Email to hiring team, Slack notification, ATS update

4. ATS Integration - Create Candidate in Greenhouse

POST https://harvest.greenhouse.io/v1/candidates
curl -X POST https://harvest.greenhouse.io/v1/candidates \ -H "Authorization: Basic YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "first_name": "Alice", "last_name": "Johnson", "email_addresses": [{"value": "alice@example.com", "type": "personal"}], "phone_numbers": [{"value": "+1-555-0123", "type": "mobile"}], "addresses": [{"value": "San Francisco, CA", "type": "personal"}], "custom_fields": { "years_of_experience": 5, "technical_skills": "Python, JavaScript, React, AWS", "screening_score": 92 } }'

🤖 Zapier Workflow Setup

ZAP #1: Form Submission → Resume Parsing

Trigger: Google Forms → New Form Response
Action: CandidateZip → Parse Resume
Field Mapping:
• Candidate Name: [Form: Full Name]
• Document URL: [Form: Resume Attachment]
• Return Fields: ✓ All (name, email, phone, location, experience, education, skills, certifications, salary)

ZAP #2: Screening Logic → Decision Router

Trigger: Webhooks by Zapier → Catch Raw JSON
Filters:
• IF experience_years >= [your minimum]
• AND skills contains ANY [your required skills]
• AND education includes Bachelor's+
Actions (Multiple Paths):
• Path A (Score ≥85): Send interview invite email + Slack notification
• Path B (Score 65-84): Add to review queue in Google Sheets
• Path C (Score <65): Log for manual review

ZAP #3: Strong Pass → Send Interview Request

Trigger: Google Sheets → New Row in "Strong Candidates"
Actions:
• Gmail: Send interview invitation email
• Slack: Post to #hiring-strong-candidates
• Greenhouse: Create candidate + application

ZAP #4: All Submissions → Analytics Logging

Trigger: Webhooks → Catch Raw JSON
Action: Google Sheets → Create Row
Logged Data: Timestamp, candidate name, email, position, score, recommendation, status

ZAP #5: Failed Parses → Manual Review Alert

Trigger: Webhooks → JSON with parse_success: false
Actions:
• Email recruiting team with resume URL
• Create row in "Failed Parses" sheet for manual extraction

ZAP #6: Daily Digest → Slack Summary

Trigger: Schedule → Daily at 9:00 AM
Action: Google Sheets → Look Up Multiple Rows
Slack Message: Total submissions, strong pass count, pass count, avg score, top positions

Installation Steps

  1. Enable Apps in Zapier: Google Forms, CandidateZip, Webhooks, Google Sheets, Gmail, Slack
  2. Create Zap #1: Forms → CandidateZip Parser (one Zap)
  3. Create Zap #2: Screening logic with decision router (one Zap)
  4. Create Zap #3-6: Notifications and logging (one Zap each)
  5. Test each Zap: Submit sample form, verify data flows
  6. Enable all Zaps when confident
💡 Tip: Use Zapier's "Code" action for complex screening formulas. Reference the Python example in the Test Tool tab.
